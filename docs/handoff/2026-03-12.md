## 핵심 결정 & 맥락
- Lv.1(세션)을 건너뛰고 Lv.2(JWT)로 직행 — 구현 난이도 차이 미미, FastAPI 생태계 호환
- Access Token 30분 / Refresh Token 30일 / Rotation 적용
- Access Token은 메모리(React state), Refresh Token은 httpOnly Cookie (path=/api/auth)
- 회원가입은 공개 안 함 — admin이 POST /api/auth/users로 사용자 생성
- OAuth(Lv.3)는 향후 검토
- DB는 SQLite로 시작, Docker 도입 후 PostgreSQL 전환 가능 (SQLAlchemy ORM이므로 코드 변경 최소)
- `syops_password` 환경변수 제거됨 — VPS .env에서도 제거 필요
