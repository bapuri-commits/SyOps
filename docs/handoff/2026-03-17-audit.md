## 핵심 결정 & 맥락
- 시크릿 키 검증: sqlite(로컬)이면 경고만, 그 외면 시작 차단 — 개발 편의성과 보안 균형
- Refresh Token TOCTOU: `UPDATE ... WHERE revoked=False RETURNING` — SQLite 3.35+ 필요 (Python 3.12 번들은 3.40+이므로 문제없음)
- `get_current_user`에 쿠키 폴백 추가: `auto_error=False` + Bearer 우선 → 쿠키 폴백
- Voca_Drill AuthContext: SyOps 패턴 따라감 (메모리 토큰, refresh 뮤텍스, 실패 시 SyOps 로그인 리다이렉트)
- AVAILABLE_SERVICES 통합: registry.py에 `access`/`data_scope` 필드 추가, `get_auth_services()` / `get_private_service_ids()` 함수로 단일 소스 달성
